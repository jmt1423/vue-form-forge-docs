## Get started

<CardGrid stagger>
  <Card title="Install source files" icon="download">
    <span class="flex flex-row items-center justify-between">
      Run `npx @vue-form-forge/cli@latest` to get the source files and
      dependencies.
      <a href="https://github.com/jmt1423/vue-form-forge" class="w-10 h-10">
        <Icon name={"external"} size="2rem" />
      </a>
    </span>
  </Card>
  <Card title="Setup TailwindV4" icon="add-document">
    <span class="flex flex-row items-center justify-between">
      Install TailwindV4 and add the `tailwind.config.js` to your project root.
      <a href="https://github.com/jmt1423/vue-form-forge" class="w-10 h-10">
        <Icon name={"external"} size="2rem" />
      </a>
    </span>
  </Card>
  <Card title="Configure Formkit theming" icon="setting">
    <span class="flex flex-row items-center justify-between">
      Add the `formkit.config.ts` and `formkit.theme.ts` files.
      <a href="https://github.com/jmt1423/vue-form-forge" class="w-10 h-10">
        <Icon name={"external"} size="2rem" />
      </a>
    </span>
  </Card>
  <Card title="Add the component anywhere" icon="open-book">
    <span class="flex flex-row items-center justify-between">
      Add the `<BuilderProvider /> and <FormBuilder />` components to your app.
      <a href="https://github.com/jmt1423/vue-form-forge" class="w-10 h-10">
        <Icon name={"external"} size="2rem" />
      </a>
    </span>
  </Card>
</CardGrid>
